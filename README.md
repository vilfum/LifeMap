\# 🗺️ КАРТА ЖИЗНИ - Life Mapping System #



\## 📋 О проекте ##

Персональная система для визуализации и организации всех сфер жизни в виде интерактивной карты.



\## 🚀 Текущий статус ##

\*\*Альфа-версия 1.0\*\* - активная разработка



\## ✨ Возможности (планируемые)##

\- 📊 Иерархическая визуализация жизни

\- 📝 Редактор контента с форматированием

\- ✅ Списки дел с отслеживанием прогресса

\- 📅 Даты и напоминания

\- 📎 Прикрепление файлов и изображений

\- 🎨 Темная/светлая тема

\- 🔍 Поиск по всей карте

\- 📊 Статистика и аналитика



\## 🛠️ Технологии ##

\- Python 3.9+

\- PyQt6 для интерфейса

\- SQLite с шифрованием

\- Windows 7/10/11



\## 📦 Установка для разработки ##

1\. Установите Python 3.9+ с добавлением в PATH

2\. Установите зависимости:

&nbsp;  ```bash

&nbsp;  pip install PyQt6 pycryptodome reportlab python-dateutil
```
