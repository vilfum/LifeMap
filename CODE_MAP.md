# LifeMap — CODE_MAP

Документ описывает структуру кода проекта: файлы, классы, функции и зоны ответственности.
Поддерживай его актуальным по мере развития проекта.

---

## `main.py`
**Тип:** точка входа

### Ответственность
- создание QApplication
- инициализация главного окна
- запуск event loop

---

## `start.py`
**Тип:** альтернативный запуск

### Ответственность
- подготовка окружения
- возможная логика инициализации перед стартом UI

---

## `simple_main.py`
**Тип:** тестовый запуск

### Ответственность
- быстрый старт приложения
- отладка UI без полной бизнес-логики

---

## `setup.py`
**Тип:** инфраструктура / сборка

### Ответственность
- описание зависимостей
- упаковка приложения

---

## `database.py`
**Тип:** слой хранения данных

### Ответственность
- подключение к SQLite
- шифрование базы данных
- сохранение и загрузка данных графа

### Архитектурные заметки
- не должен использоваться напрямую из UI
- кандидат на выделение в `storage/`

---

## `models.py`
**Тип:** бизнес-модели

### Ответственность
- описание сущностей (узлы, связи)
- структура графа
- бизнес-правила предметной области

### Архитектурные заметки
- не должен зависеть от PyQt
- ядро бизнес-логики проекта

---

## `ui_main_window.py`
**Тип:** UI (QMainWindow)

### Ответственность
- главное окно приложения
- меню и тулбары
- инициализация графической сцены
- реакция на действия пользователя

---

## `ui_graph_scene.py`
**Тип:** UI / контроллер сцены

### Ответственность
- управление QGraphicsScene
- создание и удаление узлов и рёбер
- обработка событий мыши (drag, zoom, select)

### Архитектурные заметки
- содержит смешение UI и бизнес-логики
- первый кандидат на вынос логики в сервисы

---

## `ui_node_item.py`
**Тип:** UI (QGraphicsItem)

### Ответственность
- визуализация узла
- обработка событий мыши
- отображение состояний (выделен, перетаскивается, свернут)

### Архитектурные заметки
- не должен хранить бизнес-данные
- должен ссылаться на модель узла

---

## `ui_edge_item.py`
**Тип:** UI (QGraphicsItem)

### Ответственность
- визуализация связи между узлами
- обновление геометрии при перемещении узлов

---

## `test_python.py`
**Тип:** экспериментальный файл

### Ответственность
- тестирование идей
- временный код

---

## Итоговые принципы
- UI отвечает за отображение и события
- бизнес-логика — за правила и операции
- хранение данных — отдельный слой
- CODE_MAP.md — живой документ
