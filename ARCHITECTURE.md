# LifeMap — архитектура, бизнес-логика и код

## 1. Назначение проекта
**LifeMap** — настольное приложение на **Python 3.9+** с графическим интерфейсом **PyQt6** и хранилищем данных **SQLite (с шифрованием)**.

Проект предназначен для визуальной работы с графом:
- **узлы** — сущности предметной области
- **рёбра** — связи между ними

Проект находится на ранней стадии и рассчитан на рост.

---

## 2. Текущая архитектура (as-is)

Архитектура сейчас носит **монолитный характер** с разделением по файлам:

```
UI (PyQt6)
 ├─ QMainWindow
 ├─ QGraphicsScene
 ├─ QGraphicsItem (Node / Edge)
 │
 ├─ models.py        (модели данных)
 ├─ database.py      (SQLite + шифрование)
 │
 └─ main / start     (точки входа)
```
